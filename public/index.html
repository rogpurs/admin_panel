使い方ガイド

よくあるエラーと対処

  • Host key verification failed: repo URLをHTTPSに変更するか、ラズパイにGitHub SSH鍵を登録。
  • Remote branch main not found: ブランチを `master` に変更。未指定なら自動で `main/master` を判定。
  • sudo権限エラー: `scripts/*.sh` 実行用に `sudoers` の最小権限を付与。
  • 502: 対象サービス停止、または nginx 設定競合。ジョブログと `systemctl status` を確認。
