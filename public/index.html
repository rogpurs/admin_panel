使い方ガイド

よくあるエラーと対処

  • project repo not found: 先に `初期構築` を実行してください。
  • Host key verification failed: repo URLをHTTPSに変更、またはラズパイへSSH鍵登録。
  • Remote branch main not found: `master` を指定、または自動判定を利用。
  • sudo権限エラー: `scripts/*.sh` 実行用の `sudoers` 設定が必要。
  • 502: 対象サービス停止 or nginx設定競合。ジョブログ + `systemctl status` を確認。
